All open data is good, but some is better than others. This tool gives you a certificate that will help people understand the open data you are publishing.

Help

Questions marked with a label must be answered to gain that level of the certificate.

Click on question marks or labels to find out more about what to enter.

Basic
This data satisfies the fundamental requirements for open data.
Pilot
This data satisfies the requirements for initial forays into publishing open data. It provides sufficient support to prompt some reuser experimentation with a data.
Standard
This data satisfies the requirements for routinely published open data that encourages reuse.
Established
This data satisfies the requirements for key information infrastructure.

Privacy
PilotDescribe what the people whose data is contained in this dataset were told about how the data will be used, when the data was collected.

Pilot To comply with data protection laws, data has to be used consistently with how individuals were told the data would be used. You should inform potential reusers about what that entails.

PilotDescribe the processing by which the data has been anonymised to increase the difficulty of identifying individuals from the data.

Pilot Both personal data and aggregate data should be anonymised to ensure that the likelihood of discovering information about an individual is minimised.

PilotA Privacy Impact Assessment is a process for identifying privacy risks to individuals in the collection, use and disclosure of information.

Pilot When datasets contain personal data, a privacy impact assessment should be carried out to identify risks of releasing the data, mitigations against those risks, and to put in place processes that ensure that any breaches are handled effectively.
